    🚀 Erste Schritte

    Im Tab Scannen eine Figur oder ein Set fotografieren und den richtigen Treffer antippen – oder unten per Name/Nummer suchen (reine Nummern wie 75154 oder sw0815 findet die App als Set und Figur). Mit ＋ Zur Sammlung landet der Fund nach einer kurzen Zustandsfrage in eurer Sammlung. Alles, was die Erkennung nicht kennt, geht über ✏️ Manuell erfassen.

    🔑 API-Schlüssel besorgen

    Ohne Schlüssel funktioniert das Scannen (Brickognize ist frei) – Preise, Namenssuche und Set-Inhalte brauchen aber eigene, kostenlose API-Zugänge. Eintragen kann sie nur der Admin unter Mehr → API-Schlüssel.

    BrickLink (Preise & Set-Inhalte):

    1. Ein BrickLink-Konto mit eröffnetem Shop wird benötigt – die Store-API steht nur Verkäufern offen. Der Shop muss nichts anbieten, er muss nur eingerichtet sein (BrickLink: My Store).
    2. Auf der Seite API → Register Consumer die Nutzungsbedingungen akzeptieren und einen Access Token erzeugen. Dabei beide IP-Felder (IP-Adresse und Maske) mit 0.0.0.0 ausfüllen = Zugriff von beliebiger Adresse.
    3. Die vier Werte – Consumer Key, Consumer Secret, Token, Token Secret – hier in die vier Felder kopieren und speichern.

    Rebrickable (Suche nach Namen):

    1. Kostenloses Konto auf rebrickable.com anlegen.
    2. Unter Account → API einen Key erzeugen und ins Feld Rebrickable Key eintragen.

    Danach „Verbindung testen" drücken – zwei grüne Haken, und die Preise laufen ab sofort automatisch ein.

    👤 Rollen & Rechte

    Standard: Sammlung, Wünsche und Statistik – bewusst einfach. Einkaufslisten erscheinen nur, wenn eine aktive existiert; angekommene Artikel darf jeder verbuchen.
    Sammlerprofi 💼: zusätzlich Kaufpreise & Gewinn, Einkaufslisten verwalten, Gesamtangebot, Verkaufsliste, Archiv, CSV-Import. Die Rolle vergibt der Admin in der Benutzerverwaltung („Profi"-Knopf).
    Admin 🔧: Benutzer, API-Schlüssel, Sicherung.

    💼 Flohmarkt-Ablauf (Sammlerprofi)
    1. Im Tab Listen eine Einkaufsliste anlegen.
    2. Am Stand scannen und per 🛒 Liste befüllen – die Kopfzeile zeigt den Marktwert, der Zustand ist je Artikel umschaltbar.
    3. 💰 Gesamtangebot: Der rote Vorschlag ist deine Verhandlungsbasis; „Verteilen" legt den Endpreis anteilig nach Marktwert auf die Artikel um.
    4. Zu Hause verbucht jeder die Funde mit ✔ Da! – ist alles drin, archiviert sich die Liste selbst.
    🔣 Symbole

    ⚙️ Kaufpreis automatisch (BrickLink-Ø, Datum im Tooltip) · ✏️ Kaufpreis manuell · 👥 3/4 ✔ Set-Figuren vorhanden/komplett · 📦 „steckt in euren Sets" bzw. Archiv · ⭐ auf der Wunschliste · gelb hinterlegter Set-Link mit ✔ = dieses Set besitzt ihr · ausgegraut = Listen-Artikel schon verbucht.

    💾 Sicherung & Updates

    Der Admin lädt unter Sicherung eine JSON-Datei mit allen Daten herunter und kann sie jederzeit wieder einspielen – ideal vor Updates oder für den Server-Umzug. Alternativ genügt es, die Datei data/brickfolio.db auf dem Server zu kopieren.
